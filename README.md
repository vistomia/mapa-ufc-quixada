# Mapa UFC Quixadá

Um mapa interativo e moderno do campus da UFC em Quixadá, desenvolvido com base no mapa oficial disponibilizado pela UFC. 

**Mapa oficial de referência:** [UFC INFRA - Campus Quixadá](https://ufcinfra.ufc.br/pt/mapas/)

https://vistomia.github.io/mapa-ufc-quixada/

## Funcionalidades Poggers

- Google Maps-like
